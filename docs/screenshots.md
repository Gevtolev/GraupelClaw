## 5-9. 创建 Agent 向导（4 步流程）

### 5. 第 0 步：选择起点（模板）

6 个模板可选：空白智能体、日常助手、Shopify 运营助手、Dropshipping 助手、编程助手、电商专家。可从空白画布开始，也可基于预设模板快速创建。

![[screenshot/Screenshot 2026-04-15 161516.png]]

### 6. 第 1/4 步：身份与模型

定义 Agent 基本信息：名称、头像（像意风 / 冒险家 / 机器人 / 高雅黑等风格，支持自定义上传）、描述、风格选择。右侧实时预览 Agent 卡片效果。

![[screenshot/Screenshot 2026-04-15 161525.png]]

### 7. 第 2/4 步：工具配置

8 大工具类别，每类可独立开关：

| 类别 | 包含工具 |
|---|---|
| 文件系统 | List, Read, Grep, Glob, Ripgrep, Write, Edit |
| 网络与浏览 | Web Search, Web Fetch |
| 选品找商 | Product Supplier Search |
| 代码与终端 | Bash, Process, Cron, Question |
| 图像与媒体 | Image Generate, Image Edit, See Image |
| 实用工具 | Weather, Time, Location, Listen Emails, Unlisten Emails |
| 记忆与规划 | Memory Search, Memory Get, Task Create, Task Get, Task Update, Task List |
| 智能体协作 | Sessions Spawn, Sessions List, Sessions History, Sessions Send |

底部还有"默认工作区"可选配置。

![[screenshot/Screenshot 2026-04-15 161545.png]]

### 8. 第 3/4 步：技能配置

从技能目录中选装技能，6 大类别共 80 个技能：

| 类别 | 技能数 |
|---|---|
| 货源与选品 | 9 |
| 市场调研与分析 | 16 |
| 流量获取与广告 | 16 |
| 内容创作与营销 | 17 |
| 数据分析与财务 | 8 |
| 客户生命周期与留存 | 14 |

模板推荐的技能会自动预装，其余可从目录中手动添加。

![[screenshot/Screenshot 2026-04-15 161558.png]]

### 9. 第 4/4 步：用户信息

告诉 Agent 关于用户的个性化信息：称呼（昵称）、偏好语言（默认中文）、备注（城市/职业等）、补充背景（职业/项目/偏好）。点击"完成并启动"即创建并进入对话。

![[screenshot/Screenshot 2026-04-15 161611.png]]
